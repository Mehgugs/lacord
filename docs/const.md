## lacord.const

This module contains static configuration.

#### *string* `version`

The version of lacord.

#### *string* `homepage`

The lacord project homepage.

#### *integer* `api_version`

The version of the discord api being used.

#### *boolean* `use_cjson`

Whether [cjson][cjson] or [dkjson][dksjon] is being used.
